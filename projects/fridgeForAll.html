Fridge For All:

A Mutual Aid Application

About The Project

Fridge for all was created to help manage the inventory of Philadelphia's community fridges. Over the course of the pandemic, roughly 25 community fridges and free food pantries have sprung up to help address food insecurity in the city. My group of collaborators and I created this project as a prototype to help manange the inventory of each community fridge. The application allows each user to create an account, and on the main page, to add and subtract quantities from existing fridge inventory, as well as to input entierly new items into the inventory along with a count of how many they are putting in.
