Recent Projects

Projects That I Love

The three projects below were created in collaboration with groups of colleagues from the full-stack developer course at Upenn. These projects gave us the opportunitity to work collaboatively on building applications, and fine-tune the process of integrating code from multiple developers on github.

Fridge For All

An inventory tracking application to help community fridges in Philadelphia. With this interactive app, both donors and people collecting food from the fridge can easily adjust the count of a given item.

More about the project

Happy Trails

A hike planning app, enabling you to search for area trails based on distance from your location, and desired length of your hike. Searches generate a list of trails matching input criteria, and users are able to save their favorite trails.

More about the project

Shop & Chef

An application which allows users to easily create shopping lists, search for recipes, and add recipe ingredients to their lists.
Users create a unique account and are able to save their favorite recipes.
